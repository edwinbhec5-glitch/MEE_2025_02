1 Introducción a R

1.1 Bienvenida: ¿Por qué usar R?

R es una herramienta libre, potente y reproducible para el análisis de datos en políticas públicas.
A diferencia de los programas tradicionales, R permite documentar y automatizar todo el proceso analítico, lo que mejora la transparencia y facilita la replicación de resultados por parte de equipos técnicos o auditores.

Ventajas principales:

  • Software libre y ampliamente utilizado en gobiernos, academia y organismos internacionales (BID, CEPAL, OCDE).
  • Ideal para el trabajo con encuestas, bases administrativas y microdatos.
  • Integración con herramientas de reporte (Quarto, Markdown, GitHub, etc.).

📘 Durante este módulo, R será la herramienta central para todas las etapas del curso: desde la estadística descriptiva hasta los modelos logit y probit.

1.2 Primeros pasos en RStudio

RStudio es el entorno más usado para trabajar con R.
Consta de cuatro paneles principales: 1. Script / Editor: donde escribes tu código. 2. Consola: donde se ejecuta el código. 3. Environment: donde ves los objetos cargados. 4. Files / Plots / Help: exploración de archivos, gráficos y ayuda.

1.2.2 Vectores

Es la unidad básica de almacenamiento de datos. Es una colección ordenada de valores del mismo tipo. Todos los elementos de un vector deben ser del mismo tipo de dato (numérico, lógico, carácter, etc.).

1:5
[1] 1 2 3 4 5
2^(1:5)
[1]  2  4  8 16 32
x <- 1:5
2^x
[1]  2  4  8 16 32

1.2.2.1 Vectores dobles

dbl_vec <- c(3.5, 2, -1)

1.2.2.2 Vector entero

int_vec <- c(3L, 7L, 1L)
vec <- c(3, 7, 1)

1.2.3 Factores

Es un vector especial que almacena valores categóricos, es decir, variables que toman un número limitado de categorías (sexo, área, provincia, etc.).

vec <- c(1, 2, 2, 1, 2, 1, 2)
fac <- factor(vec, levels=c(1,2), labels = c("Femenino", "Masculino"))

1.2.4 Matrices

Una matriz en R es un objeto bidimensional que organiza los datos en filas y columnas, pero con una restricción importante: todos los elementos deben ser del mismo tipo (numéricos, lógicos o de caracteres).

matrix_example <- matrix(0, ncol=6, nrow=3) #matriz nula
m <- matrix(1:12, nrow=3, ncol=4)
m <- matrix(1:12,3,4) #escribe automatico del 1 al 12
m <- matrix(c(1,2,3,4,5,6,7,8),2,4)

1.2.4.1 Para ver atributos de la matriz

attributes(m)
$dim
[1] 2 4

1.2.4.2 Para seleccionar elementos de una matriz

m[,c(2,4)]
     [,1] [,2]
[1,]    3    7
[2,]    4    8

1.2.4.3 Multiplicar matrices

m %*% matrix(1, nrow=4, ncol=1)
     [,1]
[1,]   16
[2,]   20

1.2.5 Data Frame

Un data frame es una tabla bidimensional donde cada columna representa una variable y cada fila representa una observación (por ejemplo, un individuo, hogar, empresa, etc.). Es la forma en que R organiza los datos tipo “base de datos”

# crea vectores columna g1 y g2
m1 <- data.frame(g1=c(1,2,3,4), g2=c(10,20,30,40))

# crea vectores columna ciudad, peso, altura
bdd0 <- data.frame(ciudad = c("quito", "ambato", "ibarra"),
                   peso = c(210, 150, 182),
                   altura = c(179, 170, 181))

1.2.5.1 Explorar el Data Frame

Para explorar un data frame hay que poner el nombre del objeto seguido del símbolo “$” y aparecerán todas las variables que existen en la base de datos. Seleccionamos la que nos interesa y podemos analizar y/o hacer operaciones con esa variable (vector).

bdd0$peso
[1] 210 150 182
bdd0$color
NULL
bdd0$peso + 10 #aumenta el peso en 10
[1] 220 160 192

1.2.6 Listas

Una lista es una estructura de datos muy flexible que puede contener elementos de distintos tipos y longitudes dentro del mismo objeto. Es uno de los tipos de datos más poderosos del lenguaje, porque permite agrupar cualquier cosa: vectores, data frames, funciones, resultados de modelos

lst <- list(c(1, 2), c(TRUE), c("a", "b", "c"))
